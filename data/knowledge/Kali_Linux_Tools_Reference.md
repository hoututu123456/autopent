# Kali Linux Tools Official Documentation Reference

## 资源概述
**URL**: `https://www.kali.org/tools/`

这是 Kali Linux 官方维护的工具文档库，包含了 Kali 发行版中所有预装工具的详细说明、命令参数详解和使用示例。它是渗透测试过程中最权威的工具参考手册。

## 包含内容
该网站涵盖了以下类别的工具：
- **Information Gathering (信息收集)**:如 Nmap, Recon-ng, Dmitry, TheHarvester 等。
- **Vulnerability Analysis (漏洞分析)**: 如 Nikto, OpenVAS, SQLMap 等。
- **Web Application Analysis (Web 应用分析)**: 如 Burp Suite, WPScan, Commix 等。
- **Database Assessment (数据库评估)**: 如 SQLMap, SQLite, HexorBase 等。
- **Password Attacks (密码攻击)**: 如 Hydra, John the Ripper, Hashcat, Medusa 等。
- **Wireless Attacks (无线攻击)**: 如 Aircrack-ng, Kismet, Wifite 等。
- **Reverse Engineering (逆向工程)**: 如 Ghidra, OllyDbg, Radare2 等。
- **Exploitation Tools (漏洞利用)**: 如 Metasploit Framework, Searchsploit, BeEF 等。
- **Sniffing & Spoofing (嗅探与欺骗)**: 如 Wireshark, Ettercap, Bettercap, Responder 等。
- **Post Exploitation (后渗透)**: 如 Mimikatz, Powersploit, Weevely 等。
- **Forensics (取证)**: 如 Autopsy, Binwalk, Foremost 等。
- **Reporting Tools (报告工具)**: 如 Dradis, Faraday 等。

## 如何使用
当你不确定某个工具的具体参数或用法时，请务必查阅此资源。

**最佳搜索策略**:
使用 `web_search` 工具，查询语句格式为：
`site:kali.org/tools <工具名称> usage`
或
`site:kali.org/tools <工具名称> examples`

**示例**:
- 查询 Nmap 扫描参数: `site:kali.org/tools nmap cheat sheet`
- 查询 SQLMap 注入示例: `site:kali.org/tools sqlmap examples`
